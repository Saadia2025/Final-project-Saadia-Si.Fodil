Project Name :

ETL Pipeline With API and Nodejs

Author :

Saadia Si Fodil

Project Scenario

You have been hired as a Data Engineer by a research organization. You should download the CSV file and apply ETL methods using Python. At the end, display the results after ETL in a table. Additionally, include a query to show the richest countries, for example, those with a GDP greater than or equal to 10,000 billion dollars. The final processed dataset should be stored and displayed as a database table on the API web server.

Directions:

  1. ETL with Python - create api.py

    • Use Pandas to create DataFrame
    • Use SQLite to connect to database
  1. Create app.js for API Web Server with Express
    • Serve static files (HTML, CSS, JS)
    • Create the routes
    • Start the server
  1. Create index.html
    • Use Flexbox to display two tables side by side
    • First table: data after ETL
    • Second table: data with threshold
    • Add CSS styling
